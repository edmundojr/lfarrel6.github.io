BLE Proxying

I built a system for remote access to BLE devices.

Using Noble, and Node CoAP I made a CoAP proxy server which tracks local bluetooth devices.
I also made a http client to speak to the CoAP proxy, providing users with a GUI while interacting with the proxy.

Check out the project on Github.
